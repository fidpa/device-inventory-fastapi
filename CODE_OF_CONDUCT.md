# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) version 2.1.

By participating in this project — as a contributor, maintainer, issue reporter, or commenter — you agree to follow it.

## Reporting

To report a concern, open a [GitHub Discussion](https://github.com/fidpa/device-inventory-fastapi/discussions) or contact the maintainers via [GitHub Issues](https://github.com/fidpa/device-inventory-fastapi/issues). All reports are handled confidentially.
